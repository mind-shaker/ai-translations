# Переклад через ChatGPT

Цей скілл (воркфлоу) використовує `browser_subagent` для того, щоб відкрити ChatGPT у браузері, ввести текст для перекладу та отримати результат.

## Як використовувати:
Щоб запустити цей скілл, просто попросіть мене: "Використай скілл перекладу ChatGPT для тексту: [ваш текст]".

## Кроки (для агента):
1. Отримати текст для перекладу від користувача.
2. Викликати інструмент `browser_subagent` з наступним завданням:
   - Відкрити `https://chatgpt.com/`
   - Знайти поле вводу повідомлення (prompt textarea).
   - Ввести текст: "Translate the following text into English: [ТЕКСТ]"
   - Натиснути кнопку відправки.
   - Дочекатися завершення генерації відповіді.
   - Прочитати та повернути фінальний перекладений текст.
