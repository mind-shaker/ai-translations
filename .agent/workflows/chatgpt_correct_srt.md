# Виправлення SRT через ChatGPT

Цей скілл (воркфлоу) використовує `browser_subagent` для того, щоб відкрити ChatGPT у браузері, передати йому текст у форматі SRT та попросити виправити всі помилки розпізнавання мови (особливо специфічні терміни), зберігаючи при цьому оригінальні таймкоди та формат файлу.

## Як використовувати:
Щоб запустити цей скілл, просто попросіть мене: "Використай скілл виправлення субтитрів для наступного тексту: [ваш SRT текст]". Або просто передайте текст субтитрів із проханням виправити його через ChatGPT.

## Кроки (для агента):
1. Отримати фрагмент тексту у форматі SRT від користувача.
2. Викликати інструмент `browser_subagent` з наступним завданням:
   - Відкрити `https://chatgpt.com/`
   - Знайти поле вводу повідомлення (prompt textarea).
   - Ввести наступний промпт (можна адаптувати під конкретні помилки користувача):
     ```
     Виправ контекстні неточності, одруківки та неправильно розпізнані слова в цьому SRT-файлі (наприклад, "вчачі піті" -> "ChatGPT", "адліцької" -> "англійської", "промд" -> "промпт" тощо). 
     Зроби текст максимально природним і правильним, але КРИТИЧНО ВАЖЛИВО: збережи оригінальні індекси та таймкоди без жодних змін! Не додавай своїх коментарів, просто поверни виправлений текст у форматі SRT.
     
     Ось текст:
     [ТЕКСТ_КОРИСТУВАЧА]
     ```
   - Натиснути кнопку відправки.
   - Дочекатися завершення генерації відповіді.
   - Прочитати та повернути фінальний виправлений SRT текст.
3. Отримавши результат від сабагента, надати виправлений текст користувачеві або зберегти його у вказаний файл.
